Static File Serving

Efficient serving of static resources with automatic MIME type detection

WebSrv efficiently serves static files like HTML pages, stylesheets, scripts, and media. The server automatically detects file types and sets appropriate MIME types based on file extensions.

Example Request

GET /assets/style.css HTTP/1.1
Host: localhost:8080
Connection: keep-alive

Example Response

HTTP/1.1 200 OK
Content-Type: text/css
Content-Length: 1024
Connection: keep-alive

/* CSS content here */
body { font-family: sans-serif; }

Key Features

  • Automatic MIME type detection based on file extensions
  • Support for HTML, CSS, JavaScript, images, videos, and more
  • Directory listing with autoindex on
  • Default index file serving (index.html)
  • Efficient buffered reading and writing
  • Proper Content-Length header calculation
  • Non-blocking I/O for optimal performance
